# 📱 MERN Social Media App

A simple social media web application built using the MERN stack where users can create posts and view all posts on a dedicated feed page.

🚀 Built as a full-stack project to demonstrate MERN development skills

---

## 🛠️ Tech Stack

- Frontend: React.js (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- Other Tools: Axios, REST APIs

---

## ✨ Features

- 📝 Create new posts  
- 📃 View all posts on feed page  
- 🔄 Instant update after post creation  
- 🌐 Full-stack MERN architecture  
- 📦 REST API integration
